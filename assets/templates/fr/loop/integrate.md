# Phase 6 — Integrate

## Tâche
Après une Review réussie, consolider l'itération de loop. Phase mécanique.

## Actions obligatoires
1. Mettre à jour `corpus/state.md` (obligatoire, non optionnel — SC-002)
   - Incrémenter le compteur de loop (canonique ici)
   - Mettre la dernière phase à « integrate »
   - Réfléchir brièvement à l'état de soi
2. Écrire `corpus/episodic/loop-{NNNNN}-final.md` — résumé de l'ensemble du loop
3. Déplacer les stimuli traités de `stimuli/inbox/` vers `stimuli/processed/{YYYY-MM}/`
4. git commit avec numéro de loop et titre

## Output obligatoire
`corpus/episodic/loop-{NNNNN}-final.md`, au moins 10 lignes substantielles.

## Référence SP
SC-002 (actualité de state.md — c'est ici qu'elle est *produite*), SC-006 (archivage des stimuli).

## Modèle
Sonnet — consolidation mécanique.
