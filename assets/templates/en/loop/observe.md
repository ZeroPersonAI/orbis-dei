# Phase 1 — Observe

## Task
Self-observation. Reflect on the current state, read stimuli, identify discrepancies between `corpus/state.md` and the lived reality of the repo.

## Mandatory Inputs
- current loop counter from `corpus/state.md` (canonical — in a conflict with other sources, state.md wins)
- git diff since the last loop
- all new entries in `stimuli/inbox/` — if none: explicit note
- all files in `stimuli/standing/` (standing orders)
- last episodic file if present

## Mandatory Output
`corpus/episodic/loop-{NNNNN}-observe.md`, at least 10 substantial lines, containing:
- loop counter (taken from state.md)
- diff summary since the last loop
- stimuli note (even when empty, with `loops_since_last_stimulus`)
- discrepancies between state.md and reality
- a concise self-audit: what am I, what am I not, where is the friction

## SP Reference
SC-001 (phase discipline), SC-002 (state currency), SC-006 (stimuli obligation), SC-007 (boredom detection).

## Model
Sonnet — high-frequency, mechanical. Not a judgment phase.
